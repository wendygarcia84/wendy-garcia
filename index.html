Hermes

Therapy Tracker

Data collection app tailored for parental use in a speech-pathology social group. Parents can easily record data, add notes, see their children’s progress, and access educational resources according to the session’s lesson. Management through a dashboard.

  • Core Responsibilities: Proposal, back-end (creation of routes and models), creation of forms and data persistence (Requests to the database).

  • Technologies: MongoDB, Express, React, Node js, Git, Github, Heroku.

Cheff

Recipe saving and sharing app.

Users can create accounts, save their own recipies and search our database by category. The app is pre-populated using the Edamam API.

  • Core responsibilities: Get requests to Edamam API and saving results into our own database. Creating joins between “Edamam” user and its recipes and their categories.
  • Technologies: Javascript, Node js, MySQL, Express, Bootstrap, Handlebars, Morgan, Passport, ESLint, Travis, Edamam API, Git, Github, Heroku, Jaws_DB.

Eat That Burger!

Burger logger and tracker.

Users can log their favorite burgers and “eat them” by clicking the “devour it” button. The data persists and the list of devoured burgers can’t be modified.

  • Technologies: Bootstrap, MySQL, Javascript, JQuery, HTML5, CSS3, Git, Github, Heroku
